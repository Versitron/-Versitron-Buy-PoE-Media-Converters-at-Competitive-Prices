## What is a PoE Media Converter?

A PoE (Power over Ethernet) media converter enables the transmission of Ethernet data and electrical power simultaneously over fiber optic links. These devices are commonly used to connect PoE-enabled equipment—such as IP cameras, wireless access points, and VoIP phones—to fiber networks. PoE media converters help extend network distance while reducing infrastructure costs and installation complexity.
